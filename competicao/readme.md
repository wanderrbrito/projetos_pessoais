# Simulador notas de uma competição

Pequeno projeto em Python que simula uma competição onde o usuário informa
o número de competidores e cinco notas de uma prova, onde serão eliminadas
a maior e menor nota, gerado a media e no final exibe todos os participantes
suas notas e a média e tambem o vencedor da competição


## Recursos utilizados:
- Python 3

## Conceitos praticados neste projeto

- Uso de dicionários para armazenar os participantes, as notas em uma lista e a média

## Motivação

APenas para testar os conhecimentos adquiridos em alguns cursos;


## Como o programa funciona

1. O usuário informa a quantidade de competidores
2. O usuário informa as cinco notas de cada participantes
3. O programa:
   - armazena a lista com as notas em uma lista.
   - elimina a maior e menor nota
   - calcula a média de cada participante
   - ordena o dicionário e exibe os participantes com suas notas e média
   - exibe o participante vencedor e sua média
   
##  Como executar

1. Clone o repositório ou baixe os arquivos.
2. Instale o Python 3, se ainda não tiver.
3. No terminal, dentro da pasta do projeto, execute: python competicao.py
