Simulador da Copa do Mundo de Futebol Masculino - 2026
Pequeno projeto em Python que simula o sorteio de grupos da copa de futebol de 2026
Recursos utilizados:
Python 3
Uso de lista e dicionários para armazenas as selecoes e os grupos
Uso le loops while e for para realizar os sorteios
Conceitos praticados neste projeto
Listas para cadastrar as selecoes e os grupos
Uso de dicionários para armazenar os grupos apos o sorteio
Como o programa funciona
As selções e grupos sao armazenados em listas
Uso do random.choice para realizar os sorteios
Uso de um dicionário para armazenar os grupos sorteados
Como executar
Clone o repositório ou baixe os arquivos.
Instale o Python 3, se ainda não tiver.
No terminal, dentro da pasta do projeto, execute: python sorteio_copa.py
